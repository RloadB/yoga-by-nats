# Yoga By Nats

Website for Natalia Bautista — yoga teacher based in Antwerp.

## Stack

- [Vite](https://vite.dev/) — build tool
- [React](https://react.dev/) — UI framework
- [TypeScript](https://www.typescriptlang.org/) — type safety
- [Tailwind CSS](https://tailwindcss.com/) — styling

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Build for production

```bash
npm run build
```

Output goes to the `dist/` folder.

## Deploy

Deployed via Vercel. Connect the GitHub repo to a new Vercel project and it will build automatically on push.
